目前实现的内容：

### 搅拌机科技
*胶水……*

下调了部分大机器的解锁时期，让上位机器在下位阶段就能投入使用

并没有删除原配方

已适配：木化工厂、石化工厂

### 大集气加强
*有点强……*

效率相当于不液化的空气处理效率

原版大集气并行数过少，Add的集气阶段解锁过晚，遂写

配方耗时基数为10秒、MV电压等级

为防止配方时间太短导致卡顿，自带批处理×10

由于大集气并行在后期略显不足，所有加强配方自带3次无损超频

100B*64*10=64KB

电压×3=功率×64=产出×64，批处理×10、基础产出100B，最终输出量640倍，最终输出速度64倍，因为有×10是批处理

### 添加去除维度限制的配方
部分物品及各星球流体钻机的维度限制

导致我的存档体积额外增加300MB

因为我要加载区块，跑图找流体

因此添加了移除相关维度限制的配方以缩减存档体积

### 流体钻机电解
*！؟虽虽？！*

因为我懒得写每种流体一个配方所以就一个维度一个配方了（库嘻嘻）

输出量计算公式：1024×超频后倍速（如MV钻机超频为1.5倍）×权重×10

效率加强：无尽流体钻机用电效率较差，MV到HV阶段默认赠送1次无损超频并额外提供1次无损超频，HV到EV阶段再提供1次无损超频，EV到ZPM阶段强制2次有损超频后耗电×4，因此针对性加强，改为完全无损超频

目前存在数值问题，按此设计仅MV到HV阶段的钻机有效率提升；尽管无尽钻机自带无损超频，但等能吃满并行时，大概率已无使用需求，所以换个HV钻机就够了

如果你真的很需要哪些流体，那你就可以使用其他钻机，更高的功率更高的产出，但我更推荐产线撕裂的无限元件
